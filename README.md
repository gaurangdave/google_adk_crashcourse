# Google ADK Crash Course

This notebook has sample code and practise problems as I learn Google ADK


## Setup Details
* Python v3.13.7

## Setup Steps

* Create `Python` virtual environment
```bash
python -m venv .venv
```

* Activate the environment
```bash
# macOS/Linux
source .venv/bin/activate
```

* Install Dependencies
```bash

pip install google-adk yfinance psutil litellm google-genai python-dotenv

```
